# Plan of Record (POR)

## Vision
Fast-paced 3D shooting game with smooth controls, engaging enemies, and satisfying combat.

## Phases

### Phase 1: Foundation (COMPLETE)
- [x] Vite + TypeScript + Three.js setup
- [x] Game loop with delta time
- [x] Player entity with WASD movement
- [x] Mouse aim with raycasting
- [x] Third-person camera with smoothing
- [x] Basic scene (runway, lighting, grid)
- [x] HMR support for development

### Phase 2: Combat Core
- [ ] Projectile system (player bullets)
- [ ] Enemy spawning system
- [ ] Basic enemy type (linear movement)
- [ ] Collision detection
- [ ] Health/damage system
- [ ] Enemy destruction effects

### Phase 3: Game Flow
- [ ] Score system
- [ ] Wave progression
- [ ] UI overlay (score, health)
- [ ] Game over state
- [ ] Restart mechanism

### Phase 4: Polish
- [ ] Additional enemy types
- [ ] Power-ups
- [ ] Particle effects
- [ ] Sound effects
- [ ] Background music
- [ ] Mobile controls

### Phase 5: Distribution
- [ ] Performance optimization
- [ ] Progressive Web App
- [ ] Leaderboard (optional)
- [ ] Analytics

## Current Focus
Phase 2: Combat Core - Implement shooting mechanics and basic enemies.

## Key Metrics
- Target: 60 FPS on mid-range devices
- Load time: < 3 seconds
- Bundle size: < 500KB (excluding assets)
